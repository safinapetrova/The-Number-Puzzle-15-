# The-Number-Puzzle-15-
Числовая головоломка "15"
Описание:Игровое поле размером 4x4 содержит плитки с числами от 1 до 15 и одну пустую клетку. Игрок должен перемещать плитки, чтобы упорядочить их по возрастанию, перемещая их на свободное место.
Особенности реализации: Двумерный массив для хранения состояния игрового поля, функции для перемещения плиток и проверки состояния игры.
Инструкция: 1.
            2.
            3.
            4.
            5.
